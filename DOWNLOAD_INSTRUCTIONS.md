# How to Download and Share This Project

## Quick Download Guide

### Option 1: Download from Replit
1. Click the three dots menu (⋯) in the top-right corner of Replit
2. Select "Download as zip"
3. Save the ZIP file to your computer
4. Share the ZIP file with your group members via:
   - Email
   - Google Drive
   - Dropbox
   - WeTransfer
   - USB drive

### Option 2: Clone with Git
If you're familiar with Git:
```bash
# Your group members can clone using:
git clone [your-replit-url]
```

## What Your Group Members Need to Do

### To View the Website:
1. **Extract the ZIP file** (if downloaded as ZIP)
2. **Open the folder** in their file explorer
3. **Double-click `index.html`** to open in their web browser
4. **Navigate** using the menu to see all 11 pages

### No Installation Required!
- No software installation needed
- No setup process
- Works on Windows, Mac, and Linux
- Compatible with Chrome, Firefox, Safari, Edge

## Important Notes for Your Group

### Before Sharing:
1. **Replace Trustee Photos**: 
   - Go to `pages/trustees.html`
   - Replace the image URLs with actual photos of your group members
   - Or edit the HTML to add local images

2. **Update Contact Information**:
   - Edit the footer in each HTML file
   - Replace placeholder contact details with real or fictional ones for your project

3. **Add Your Names**:
   - Update the README.md with your actual university name and course code
   - Add all group member names

### Customization Tips:
- **Colors**: Edit `css/style.css` (lines 4-9 have the color variables)
- **Text**: Simply edit the HTML files to change content
- **Images**: Replace Unsplash URLs with your own images
- **Fashion House Name**: Currently "Élégance Couture" - change if desired
