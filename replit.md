# Élégance Couture - Fashion House Website

## Overview
This is a complete fashion house website built exclusively with HTML, CSS, and JavaScript for a university group project. The project showcases "Élégance Couture," a luxury fashion brand, and includes all required features for the assessment task.

## Project Architecture

### Pages (11 Total)
1. **index.html** - Homepage with hero section and featured products
2. **pages/about.html** - Company story, vision, and values
3. **pages/products.html** - Complete product catalog (12 items)
4. **pages/collections.html** - Seasonal collections showcase
5. **pages/lookbook.html** - Style inspiration and outfit ideas
6. **pages/trustees.html** - Board of Trustees (6 members with photos)
7. **pages/events.html** - Upcoming events (6 fashion shows/events)
8. **pages/gallery.html** - Fashion photography gallery (12 images)
9. **pages/appointments.html** - Booking form with real-time validation
10. **pages/contact.html** - Contact information and inquiry form
11. **pages/dom-structure.html** - Website DOM hierarchy documentation

### Styling
- **css/style.css** - Single external stylesheet for all pages
- Responsive design (mobile, tablet, desktop)
- Professional color scheme (dark gray, gold, tan)
- Custom animations and transitions

### JavaScript Features
- **js/main.js** - Navigation, animations, gallery modal, back-to-top button
- **js/form-validation.js** - Real-time form validation with error messages
- Mobile menu toggle
- Active page highlighting
- Scroll-triggered fade-in effects
- Image modal viewer
- Smooth scrolling

### Assets
- **images/** - Placeholder folder for project images
- Currently using Unsplash URLs for demonstration
